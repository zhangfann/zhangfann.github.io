                    # 丢失的密码本

                    # 一 摊位老板

                    我是土生土长的青州市人, 打拼多年, 也算是攒下来一些积蓄, 受够了职场的纷纷扰扰, 刚好年轻人里面地摊经济日益兴起, 便租下家附近商城内市集的一个摊位, 平时喜欢喝茶, 便在摊位搞起了茶具生意, 每天的工作就是煮茶, 喝茶, 也算是清闲自在

                    # 二 直播

                    最近几天晚上闲来无事, 刷起了视频网站, 大主播间实在太闹腾, 推销产品的话术无孔不入, 让人难以看下去, 连着刷了几天, 一个定位青州的主播吸引了我的注意力, 主播名叫小杨, 一个刚毕业的美术专业学生, 与其他叫卖商品的主播不同, 小杨只是对她制作版画的过程进行直播,

                    小杨通常晚上8点开播, 直播中偶尔说说新进来的人的名字, 水友之间互相在那聊天, 可今天已经晚上9点了, 小杨怎么还没开始直播, 我心里纳闷,

                    半个小时后, 小杨才姗姗来迟, "谁能想到一号线晚上人那么多, 我都是双脚离地, 被人挤上车的", 小杨应该是遇到了一号线的晚高峰 "挤进车门的时候, 包里还被人塞了一本厚书, 还有意外收获呢", 小杨拿了本厚书出来, "没事没事, 我也最讨厌周一早上的地铁了", 我隔着键盘打字, 和小杨聊起来

                    是黑石一雄的<远山淡影>, 这年头抱着这么一本大书挤地铁的人, 一定是个爱书之人吧, 看着厚厚的一本书, 我内心满是respect

                    # 三 榜一大哥

                    水友投币往往是薅羊毛买什么东西之类，网站送的币, 每天总有人送上几个, 活跃气氛 这天, 直播间突然空降一个大哥, 陆陆续续刷了几百块钱 自称熟悉版画这一行, 看小杨虽然现在粉丝少, 但有潜力, 总是在闲聊的时候和小杨探讨业务。 大哥一直鼓励这个快坚持不下去的主播, 偶尔刷刷视频碰上了就来看看

                    得知我是摊位老板, 大哥也撺掇起来, 觉得要是小杨能在线下摊位卖插画, 效果肯定会更好 别说, 我的摊位旁边正好有个闲置的摊位, 大哥这想法确实靠谱, 小杨也心动起来, 线下卖插画, 能直接给小杨增加收入, 小杨的直播, 也能给我的摊位带来更多流量, 小杨立刻便准备起来

                    # 四 小杨的新摊位

                    事不宜迟, 一周后的周一，"小杨的版画商店"在我的摊位旁边隆重开幕了。 小杨的版画在市集中也算是独具一格

                    小杨的摊位很快开张起来, 并很快和左邻右舍打成一片, 小杨的左边是我, 一个卖茶具的老男人, 右边是zoey, 一个也总是迟到的心理学系毕业生, 小杨和zoey一边蹭着我的单纵, 一边聊着八卦

                    # 五 钱夫妇

                    小杨的冷摊还是受到大家的欢迎, 一男一女来逛，看着三十来岁，这对夫妇都姓钱, 小杨叫他们钱夫妇, 钱夫妇给小杨聊天, 很是喜欢小杨的版画 小杨想, 肯定是钱夫妇吃了太多鲍鱼海参官府菜, 偶尔用"萝卜咸菜"换换口味。 钱哥的眼睛特别毒, 瞟了台面一眼就说："唉，你这些，既美好，又没用" 这波点评太到位了, 小杨马上拿小纸片把句子写下来, 奉为揽客金句 印完小画都会剩下些纸头, 尤其这次调蓝墨时, 意外调出了些蓝染色, 随便印点图案就好看 钱哥当场就在一张卡片上签上了自己名字，又把笔递给自己的钱嫂签名，然后拿出一张宝丽来照片，估计刚照的，一起塞进小信封里

                    钱哥趁机又问:"如果给他们制作一副合照版画, 得多少钱" 如果只是单独定作, 版画的制作成本很高, 小杨老实回答:"至少得五千快" 钱嫂听说要五千块钱，就一声惊呼，"天呀!"还打了个冷颤! 钱嫂这时看到了小杨桌上的<远山淡影>, 又是一声惊呼:"天呀!这是初印版的<远山淡影>, 而且是原版的, 市面上非常稀有!" 钱哥愤愤不平地接话到:"现代人太不尊重知识的价值了, 这么好的书, 因为不赚钱就不印了", 小杨也不想放走这对大客户, "这说明咱们有缘分, 版画的价格还能谈", 并加了钱夫妇微信, 估计时做好了打持久战的准备, 因为加了微信，俩小时后，我和小杨看到了这位"大客户"的朋友圈，在地铁里返图<远山淡影>，一定是真喜欢吧，所以, 估计小杨还是有机会但

                    # 六 大哥的捧场

                    下午快结束时, 直播间的大哥还专门到现场捧场, 鼓励她保持下去 大哥一看 "既美好又没用" 的卡片噗嗤就笑了，咯咯咯地说"你的形容很贴切啊" 小杨老实说这是一位客人给的点评, 可能怕打击到她, 大哥一口气买了四张"没用"，成为当日最大买家

                    # 七 大哥的秘密

                    "大哥, 咱再往里面砸钱, 就得去要饭了", 小弟看着带着四张版画回来的大哥说到 "这叫放长线吊大鱼", 大哥得意地说, "咱放了半天长线, 可密码本连影子都没看见", 小弟多少有点沮丧 "你懂啥, 现代社会讲究的是格局", 大哥没好气地说,

                    不过一想起密码本, 大哥也是心痛不已, 原来大哥是米国特务, 这次的任务是将青州本地的特务带来新的密码本带回米国, 是一本错版印刷的<远山淡影>, 这本书太稀有了, 本身原版的<远山淡影>就不多, 错版的更是全世界都没有几本, 密码就是依赖错版的<远山淡影>的几处错误, 作为加密密钥 这次任务为了掩人耳目, 选择在了1号线早高峰进行交接, 但是人实在太多了, 大哥不仅没完成任务, 还把密码本搞丢了,

                    "在1号线上丢东西很正常, 我的iphone就是一个大哥打完电话, 硬塞我兜里的", 小弟晃了晃手中的iphone, 大哥又有点上头, "是谁没日没夜地查看地铁监控, 找到了那个小姑娘?", "是谁找机会把她约到线下?", "我这是在下一盘大棋!", 大哥略显得意,

                    大哥说着从兜里把<远山淡影>往桌子上一拍, "这事会在神不知鬼不觉中解决掉", 原来大哥在神不知鬼不觉中, 把小杨摊位的<远山淡影>带来回去, "嗯, 大哥英明, 不然我们得到西伯利亚种土豆去了", 小弟松了一口气

                    为了避免夜长梦多, 大哥和小弟连夜打飞的回了米国, 去向组织交差了,

                    # 八 小杨的大生意

                    钱夫妇思来想去, 觉得如果小杨能把原版的<远山淡影>送给他们, 那么版画的价格也是可以接收的, 小杨想, 书能帮助自己做成生意, 能到欣赏它的人的手中,能发挥更大的价值, 便一口答应下来 约定第二天, 先将<远山淡影>送给钱夫妇

                    # 九 尾声

                    第二天, 小杨才发现书丢了, 这可把她急坏了, 没了书, 给钱夫妇做版画估计也得泡汤, "张哥, 这可咋办啊? 能帮我搞本原版的<远山淡影>吗? 就是之前我在地铁上被人塞的那本", 小杨只好求助于我, 边说边端起我的茶壶, 猛嘬几口单纵, 让我心疼不已 "那天中午端着锅快拿不动了还找不到隔热垫就……就这样用了两天，太好用了！大不敬了，杨哥", 小杨的<远山淡影>被我拿来垫茶壶了, "之后我又在拼嘟嘟上买了一本新的, 塞到你的摊位上了, 所以丢的是盗版的, 原版的还在茶壶下面垫着",

                    小杨的第一个大生意应该是保住了, 但是直播间里的大哥却没再来过, 小杨觉得是失之东隅，收之桑榆, 也逐渐将重心从直播转移到了给人定作版画上, 只偶尔播一播, 和水友聊聊天, 慢慢也体会到版画的特质，手工&手写，仍然有存在的意义，在人来人往花花绿绿的市集上，让浪漫这个虚词落了地，具体而有型。

                    在小杨偶尔的直播中, 大哥的去向成了水友们津津乐道的话题, 传说是被派去了西伯利亚执行秘密任务了
