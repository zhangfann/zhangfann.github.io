::: 设计师制造东西, 技术是不断进步的 在技术的浪潮中 指导设计师的是 craft构建 reflection反思 value价值 :::

设计师与政治家和商业顾问有一个共同的问题：没有人知道他们究竟在做什么。我可以给大家介绍一下设计师的工作。他们塑造了我们的日常环境。如何塑造？他们制造东西。像产品、通讯或系统这样的东西，可以满足我们的需要并提供意义。 设计师使这些东西变得易懂、有用和美丽。 他们把新的想法引入到我们的日常生活中。 而这些想法是不断变化的。我们见证了科学的进步，新技术的发展，市场的混乱，不可预见的挑战的出现，以及随之而来的品味的时尚：今天我们认为是丑陋的东西，明天就是美丽的。 好的设计师跟上了这种变化，优秀的设计师推动了这种变化。 设计是进步的--一直如此。 但是，在这个变化的海洋中，使设计师保持漂浮状态的指导思想是什么呢？三件事。工艺，反思，和价值。

# craft - what do you make?

我们制造东西的方式已经发生了变化。我们开始用我们的双手。然后我们创造了工具，成为工匠。随着工业化的大规模生产，我们需要绘制新形式的产品，这些产品可以在工厂里生产并供应给消费者市场。最近，数字革命催生了网站或手机等复杂系统的设计。 从赤手空拳到复杂系统设计的演变并不意味着新方法取代了旧方法。相反，每一个发展都为我们制造东西的方式增加了一个新的层次。今天的设计仍然牢牢扎根于工艺，尽管我们对它的理解已经发生了变化。 今天，我们可以在更广泛的意义上将工艺定义为一种活动，我们为了它自己而做，以追求精湛的技艺。作为设计师，我们从我们喜欢做的东西开始。 这种工艺是我们灵感的主要来源，也是我们与世界的联系，因为我们制造东西。或者更准确地说：我们制造的东西是可行的。 但有时它们并不成功。 正是这些挫折的时刻，迫使我们学习，纠正我们对世界的想法。作为工匠，我们遇到的挫折对我们与世界的积极关系至关重要。它使我们与现实保持联系。

# reflection - how dow you make it?

::: 制造很重要 产生制造的idea的过程也很重要 ::: 制作是第一位的，但设计者仍然必须思考。 当我们改善我们的环境时，我们可能一开始就专注于解决一个问题的物体。比方说，我们想找到一个更好的方法来保存食物，于是想出了一个新的食物容器。这个物体--这个新容器--就是问题的解决方案。 但是，如果我们不停留在这个物体上，而是看看我们是如何得到这个解决方案的呢？然后我们就可以抽象出这个过程，并将其应用于其他问题。 设计师们不仅痴迷于结果，也痴迷于他们如何得到结果。因此，我们要澄清和改进创意过程。这包括常见的设计思维工具和创意技巧，但最重要的是：了解我们自己，培养我们个人的创意习惯。 创意过程是我们超越常规的混乱冲动与将非常规想法转化为实际成果所需的有序纪律的相互作用。 如上所述，设计与当前的时代潮流是同步的。在一个以如此快速的力量不断变化的职业环境中，设计师必须能够定期调整他们的行动路线。"我们到底在这里做什么？"这不仅是让单个项目保持在正轨上的核心问题，也是确保我们的活动与世界上发生的事情相一致的纠正措施。 这说起来容易做起来难，光有好奇心是不够的。要消化和理解各种想法和技术的来龙去脉，需要有分析性的思维和连接这些点的能力，以及讲述故事。

# value - for whom do you make it ?

::: 没有为他人提供价值的设计--是没有价值的。另一方面，为人们提供价值的设计师会得到价值的回报。 当然，最难的部分是找出人们想要的东西。设计师们对这个问题很着迷。 ::: 我们在日常生活中如何做事，以及我们用什么来做事，塑造了我们的身份。例如，想想我们吃什么以及我们如何准备我们的食物。我们使用的物品的功能和形式都是我们生活的社区的一部分。 作为设计师，我们在这个集体品味的领域里活动。然而，我们也超越了它，我们扩大了它。我们必须要有原创性。 要做到这一点，我们必须发展我们的技能，找到我们自己的声音。我们必须将自己与传统脱节，以遵循我们个人的痴迷和特质。 我们必须离开正常的东西。 然而，这种对个人兴趣的专注和在艺术痴迷中发现的快乐可能会成为一个滑坡。 我们可能会忘记，我们创造的东西必须融入他人的日常生活。制作原创产品是不够的，其他人必须喜欢它们。 我们必须回家。 设计师的工作是让其他人快乐。 没有为他人提供价值的设计--是没有价值的。另一方面，为人们提供价值的设计师会得到价值的回报。 当然，最难的部分是找出人们想要的东西。设计师们对这个问题很着迷。
