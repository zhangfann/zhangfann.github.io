# 一个非常「简单」的清单 APP，希望你会喜欢

这是刚刚开发完成并上架的一款产品，还很热乎，而且名字就叫做「简单」，简洁清单的意思。

它非常简单，只是一个列表而已。

因为是第一版本，所以有一些计划中的功能还没有进入开发，想先放出来，收集一下反馈。

希望有爱心且有兴趣的朋友们，可以下载体验一下，你的每一个想法和建议都对我很重要，谢谢各位。

⚠️注意：这只是一个简单的清单，并不是 TodoList，当然，未来也有可能变成 TodoList，只是当下我更想要一个简单的清单而已，不需要完成、未完成等状态。

然后，我会简单介绍一下目前它的功能。

# 操作

首次安装后，会有一个简单的指导清单，帮助你快速学习如何使用这个 APP，可以跟随它简单操作一下。

首先是添加列表项目，列表中下拉即可快速添加一条记录；点击右下角的 “+” 按钮也可以快速新增一条。

向右👉滑动可以打开删除按钮，向左👈则可以选中当前项目，而在你向左滑动的同时，如果手势向上或向下滑动，则可以快捷选择多条或取消选择。

点击每一个项目左侧的 “—” 同样可以直接选择当前项，如果在每一条项目中长按，可以快捷编辑当前项。

点击进入子列表后，使用手势向右滑动或点击顶部的 “X” 按钮都可以返回上一条。

选择进入排序模式后，可以长按列表项并拖动排序，把你关心的列表放在最上面即可。

# 功能

如你所见，整个 APP 的功能非常简单，就是一个个的列表。

每一个列表的子项目，还可以添加更多子列表，没有层级限制，想加多少层就加多少层。

为了标识出来你关心的条目，只需要轻点底部的星星按钮即可，之后它除了会在列表中有一个特殊的星星标识外，还会进入菜单中的收藏列表，可以在列表中快速查看自己收藏的条目。

每个条目还可以选择不同的自定义标签，自定义标签同样支持筛选。刚开始其实无需为列表打标签，但当你列表变得多起来之后，标签功能会变得更实用。

考虑到如果每一个列表都是清一色的黑色，可能会太单调，所以添加了选择颜色功能，目前支持的其它七种颜色都在下面。

最后要说的就是搜索了，如果你的清单变得越来越多以后，就可以用搜索来快速找到想要的清单了，菜单栏中选择搜索，输入关键词即可。

嗯，这差不多就是所有的东西了，就这样。

# 未来

就像最开始提到的一样，未来并不排除会变成 TodoList，其实开发起来也很简单，只是现阶段我不想做。

但是这些功能是会考虑在未来的版本中更新的：

  • 批量修改颜色
  • 移动清单
  • 导出为图片、文本或思维导图
  • 分享
  • 空白处快速添加条目
  • 语音输入

如果你有更多更好的想法或建议，也可以留言告诉我。

# 最后

整个 APP 不需要任何权限授权，甚至不需要网络，同时也接入了 iCloud ，可以同步所有数据到 iCloud 中备份，更棒的是跨设备同步，如果你有两台手机，两台手机的列表数据会无缝同步。

关于数据隐私问题，APP 除了 iCloud 同步外，不请求任何服务器，而如果你选择了同步到 iCloud，则只会同步到 iCloud 的 Private Database，因为 Private Database 属于苹果用户的个人私有数据库，除了你自己之外，任何其他人都没办法看到。

所以，我是不知道你同步了哪些东西到苹果服务器的。

另外，深色模式也是支持的，会随系统当前模式切换。
