    # 1. 我的 Win 实用软件清单

    排名不分先后且长期更新 有更好用的或者需要帮助的可以留言————最后一次更新于 2019.06.25

    # 1. Dism++

    # 1.1. 功能：

    系统精简、垃圾清理、系统升级、esd 转 iso、右键、启动项。。。。。功能非常多且很占用很小解压即可用！基本可替代一般清理和辅助优化软件

    # 1.2. 官网：

    Dism++：https://www.chuyu.me

    # 2. FsCapture

    # 2.1. 功能：

    各种强大的截图功能，相对一般的录屏功能，还有一些色彩拾取、尺子的小功能。最不错的是长截屏功能

    ，首屈一指。

    # 2.2. 官网：

    FsCapture (可以找到到中文版)：http://www.faststone.org/FSCapturerDownload.htm

    # 3. GifCam

    # 1.3.1. 功能：

    最简单但是各方面都是非常棒的一款 GIF 动图录制小软件，优秀。

    # 3.2. 官网：

    GifCam （下载地址在页面底部）：http://blog.bahraniapps.com/gifcam/

    # 4. Shadow*Socks

    ————文明上网

    # 5. Snipaste

    # 5.1. 功能：

    简单实用的截图功能和截图后简单的图片编辑，特色就是超好用的截图贴图功能。

    # 5.2. 官网：

    Snipaste （免费，有能力可以赞助一下）：https://zh.snipaste.com/

    # 6. Ditto

    # 6.1. 功能：

    强大的剪贴板保存，可免去来回复制粘贴的烦恼。

    # 6.2. 官网：

    软件托管地址：https://sourceforge.net/projects/ditto-cp/

    Github 项目地址：https://github.com/eclipse/ditto/releases

    # 7. QuickLook

    # 7.1. 功能：

    win 上比较不错的文件预览软件，另外有 Seer（付费）也很不错，个人比较喜欢 Quicklook。

    # 7.2. 官网：

    QuickLook：https://pooi.moe/QuickLook/

    # 8. Qtranslate

    # 8.1. 功能：

    像名字一样，翻译软件，包含 google、bing、有道、百度等等翻译 API，支持划词翻译和 ORC 识别，棒棒哒。

    # 8.2. 官网：

    QTranslate：https://quest-app.appspot.com/

    # 9. PicGo

    # 9.1. 功能：

    Win 上比较好的图床工具，mPic 据说也不错，可惜太久没更，放弃了。macOS 上 ipic 口碑不错。

    # 9.2. 官网：

    Github 地址：https://github.com/Molunerfinn/PicGo/releases

    # 10. Listary

    # 10.1. 功能：

    Win 上快速搜索文件、文件夹和应用。试过 Wox+Everything 的组合，感觉速度是比较快，但是本着少装一点的原则，选择了 Listary

    # 10.2. 官网：

    Listrary：https://www.listary.com/

    # 11. 7-Zip

    # 11.1. 功能：

    开源解压工具，都说 BandZip 更好，但是我就不喜欢，因为 Bandzip 右键不能合并在一块，而且图标又丑，右键时总感觉乱麻麻的。

    # 11.2. 官网：

    7-Zip：https://www.7-zip.org/

    # 12. Cmder

    # 12.1. 功能：

    有了它，可以弃用 cmd 和 powershell 啦，拥有近乎 Linux 系统的体验，不过需要稍微配置一下才能用。网上教程很多。

    # 12.2. 官网：

    Cmder：https://cmder.net

    # 12. Windows Terminal

    # 12.1. 功能：

    我现在用 Windows Terminal 预览版，很漂亮，配合 wsl2 链接 Ubuntu 子系统，非常好用。win 商店下载

    # 13. PuTTy

    # 13.1. 功能：

    win 上开源强大的 SSH 软件和 FileZilla 配合使用，可以代替 Xshell 和 Xftp。

    # 13.2. 官网：

    PuTTy：https://www.chiark.greenend.org.uk/~sgtatham/putty/

    # 14. FileZilla

    # 14.1. 功能：

    Xftp 替代工具，配合 PuTTy 使用。

    # 14.2. 官网：

    FileZilla：https://filezilla-project.org/

    # 15. OneDriver：

    # 15.1. 功能：

    网盘同步工具，不同于百度云之流，文件直接同步，和资源存到百度网盘这种不同，具体可自行搜索。虽然 DropBox 体验更好，但是这款不需要科 * 学上网就可以用，方便一些。

    # 15.2. 官网：

    OneDriver：https://products.office.com/zh-cn/onedrive-for-business/online-cloud-storage

    # 16. Typora

    # 16.1. 功能：

    目前用到过最舒服的 MarkDown 编辑器。谁用谁知道。

    # 16.2. 官网：

    Typora：https://typora.io/

    # 17. IDM：

    # 17.1. 功能：

    Win 上最佳下载器，最高 32 线程。喜欢折腾可试一下 Aria2，非常好。IDM 或者 Aria2 配合脚本基本可解决网盘限速问题

    # 17.2. 官网：

    IDM 下载器：http://www.internetdownloadmanager.com/

    # 18. BandiCam

    # 18.1. 功能：

    三大录屏神器之一，体验一级棒。

    # 18.2. 官网：

    BandiCam：https://www.bandicam.cn/

    # 19. VLC

    # 19.1. 功能：

    开源视频音频播放器，PotPlayer 都说很好，我一般用不到那么多功能，就这个就不错。有需要的可以试试看 PopPlayer。

    # 19.2. 官网：

    VLC：https://www.videolan.org/index.zh.html

    # 20. Xmind

    # 20.1. 功能：

    思维导图最佳选择之一。个人比较喜欢，另外几款也不错，看个人喜好。

    # 20.2. 官网：

    Xmind：https://www.xmind.net/

    # 21. WPS

    # 21.1. 功能：

    word、excel、ppt、pdf 编辑查看。个人对这些都只是基本需求，用不到 Office 那几款软件那么强大的功能，所以用这个可以完全满足所需功能。

    # 21.2. 官网：

    WPS （政府 * 特供版无广告，或者普通版安装后点卸载，卸载原因 选广告太多，可弹出关闭广告按钮。）：http://www.wps.cn/)

    # 22. ShadowDefender

    # 22.1. 功能：

    简单强大的系统还原工具，无论做了什么，重启之后一切如初。

    # 22.2. 官网：

    ShadowDefender：https://www.shadowdefender.com/

    # 22. Windows SandBox

    # 22.1 功能：

    Win10 功能里自带的 沙盒 应用，同样可满足我的需求。安装一些临时用的软件或者不信任的软件，用完之后 × 掉一切恢复如初。

    # 23. Nopad++

    # 23.1. 功能：

    文本编辑器，支持几乎所有格式，替代 win 上的记事本最佳选择。

    # 23.2. 官网：

    Notpad++：https://notepad-plus-plus.org/download/

    # 24. Thunderbird

    # 24.1. 功能：

    替代 Win10 邮箱用的，自带的其实也不错。这个开源，功能更多一点。

    # 24.2. 官网：

    Thunderbird：https://www.thunderbird.net/

    # 25. VS Code

    # 25.1. 功能：

    很优秀的。不管是 Python、Java 还是 MarkDown 文件 统统可以用插件搞定。轻编辑非常推荐，做项目有很多文件子文件的时候还是 JetBrains 完整的 IDE 更好一点

    # 25.2. 官网：

    VS Code：https://code.visualstudio.com/

    # 26. IDEA

    # 26.1. 功能：

    一款 Java IDE。虽然 VS Code 很强大，但是插件一多，或者项目文件比较多时就有点鸡肋了，IDEA 真的很好用，著名的 Android Studio 就是基于 IDEA 改的。

    # 27. Pycharm

    # 27.1. 功能：

    Python IDE，很 nice。

    IDEA 与 Pycharm 同属一家——JetBrains。官网有社区版和专业版下载链接，这里不再放出链接。有汉化包，需要的可以说一下。

    # 28. XYplorer

    # 28.1. 功能：

    Win10 文件资源管理器替代者，另外 Directory Opus 也相当不错，都是收费的，自行斟酌。

    # 28.2. 官网：

    XYplorer：https://www.xyplorer.com/

    # 29. Total Uninstall

    # 29.1. 功能：

    完全卸载软件，自带卸载程序，会残留大量文件夹和注册表垃圾，这款软件就可以很好的完全卸载程序，无残留。另外也有其他卸载软件，我个人用着就这个最舒服。（WIndows store 搜索下载）

    # 29.2. 官网：

    Total Uninstall：https://www.martau.com/uninstaller-download.php

    # 30. Windows To-Do

    # 30.1 功能：

    简单好用免费无广告多平台同步，Android、ios、win10、Mac os 无缝同步。（ 搜索下载）

    # 31.Charles

    # 31.1 功能：

    抓包分析必备。

    # 32.1 官网：

    Charles：https://www.charlesproxy.com/

    # 32.Postman

    # 32.1 功能：

    api 测试封装。

    # 32.2 官网：

    Postman : http://www.getpostman.com/

    # 33. 火绒

    # 33.1 功能：

    360 替代者，该有的功能都有，很安静不流氓。

    # 33.2 官网：

    火绒：https://www.huorong.cn/

    微信 QQ 不做解释，另外推荐 Tim 给大家代替 QQ 很不错。
