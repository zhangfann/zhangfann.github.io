        网络是如何发展的，而更重要的是--它现在要去哪里？

        另外，为什么这些都是重要的？

        假设这一系列的经验向我们展示了什么，这些进展将是关键的。在这篇文章中，我将解释网络是如何发展的，它接下来会去哪里，以及为什么它很重要。

        # Web2.0货币化和安全

        Web2世界中许多流行的应用程序在其生命周期中遵循一个共同的模式。想一想你一贯使用的一部分应用程序，以及所附的模式如何适用于它们。

        # 应用程序的货币化（针对开发者和组织）

        想象一下像Instagram、Twitter、LinkedIn或YouTube这样的流行应用程序的早期，以及它们今天的不同。

        这个过程通常看起来是这样的。

        • 该公司推出了一个应用程序

        • 纳入尽可能多的用户

        • 然后将其用户群货币化

        当一个工程师或组织提供一个著名的应用程序时，客户的体验通常是非常顺利的，因为该应用程序不断地填补突出的位置。这就是为什么他们能在任何情况下迅速获得立足点的理由。

        1. 起初，许多软件公司并不担心赚钱的问题。

        2. 他们肯定专注于增长和锁定新用户 - 但他们最终应该开始盈利。

        3. 他们还需要考虑外部投资者的作用。

        4. 通常情况下，进行风险投资等事情的限制对生命周期和最终我们今天使用的许多应用程序的用户体验有负面影响。

        如果一家应用开发公司接受了风险投资，其投资者通常期望获得数十倍或数百倍于他们所支付的投资回报。

        这意味着，该公司不是去寻求一个他们可以在某种程度上有机维持的可持续增长模式，而是经常被推到两条路上：广告或出售个人数据。

        对于一些web2组织，例如谷歌、脸书、推特等，更多的信息会促使更多的定制广告。

        这导致了更多的抓拍和最后更多的推广收入。客户信息的滥用和集中化是我们今天所知道和使用的网络打算如何工作的中心。

        # 安全和隐私

        Web2应用程序经历了反复的数据泄露。甚至有一些网站专门关注这些漏洞，让你知道你的信息何时被泄露。

        在Web2上，你无法控制你的数据或数据的存储方式。

        事实上，公司经常在未经用户同意的情况下监测和记录用户数据。

        然后所有这些数据都由负责这些平台的公司拥有和控制。

        生活在需要关注言论自由的负面后果的国家的用户也面临着风险。

        国家管理部门经常关闭服务器或扣留财务余额，假设他们接受个人发表的评估与他们的颁布相冲突。有了合并的服务器，国营行政部门很容易在他们认为合适的时候干预、控制或关闭应用程序。

        由于银行也是数字化的，而且是集中控制的，所以政府也经常在那里进行干预。在动荡、恶性通货膨胀或其他政治动荡时期，他们可能会关闭银行账户的访问或限制资金的使用。

        Web3计划通过在一般意义上重新考虑我们如何从最早的阶段开始计划和联系应用程序，来解决相当多的这些缺陷。

        # 什么是Web3.0？

        在web2和web3的范围内，有几个关键的对比，然而，去中心化是其中心。

        Web3增强了我们今天所知的互联网，并增加了一些功能。

        • 可验证的

        • 不可靠的

        • 自主的

        • 未经授权的

        • 分布式的和强大的

        • 状态信息

        • 当地居民支付

        在Web3中，设计者在大多数情况下不会构建和发送对单独的服务器需求突然激增的应用程序，或将其信息存储在单独的数据集中（一般由单独的云供应商促进和监督）。

        相反，web3应用程序要么在区块链上运行，要么在由许多点对点节点（服务器）组成的去中心化网络上运行，或者两者结合形成一个加密经济协议。这些应用程序一直被建议为dApps（去中心化的应用程序），你会注意到这个术语在Web3空间中经常被使用，这是可以想象的。

        为了完成一个稳定和安全的去中心化组织，网络成员（设计者）被敦促并争夺向所有利用帮助的人提供最大类型的援助。

        当你听到Web3时，你会意识到，加密货币往往是谈话的一部分。

        这是因为加密货币在许多这些协议中发挥着巨大的作用。

        它为任何希望参与创建、董事会、承诺或改进其中一项活动的人提供货币动力（代币）。

        这些协议通常可以提供各种不同的服务，如计算、存储、带宽、身份、托管和其他过去通常由云提供商提供的网络服务。

        个人可以通过参加公约的各种方式，在专业和非专业的层面上获得。

        服务的消费者通常为使用该协议而付费，类似于他们今天向AWS这样的云供应商付费的方式。除了Web3，这些钱都直接给了网络参与者。

        # 本地支付（成为重要的交易所）

        代币还提供了一个完全无框架和无麻烦的本地支付层。像Stripe和Paypal这样的公司在实现电子支付方面已经创造了数十亿美元。

        这些系统极其复杂，仍然不能提供参与者之间真正的国际互操作性。他们还要求你提供敏感信息和个人数据来使用它们。

        像MetaMask和Torus这样的加密货币钱包允许你在web3应用程序中协调简单、神秘和安全的全球分期付款和交换。

        像Solana这样的网络提供了几百位数的毫秒级延迟，交易成本仅为一毛钱的一小部分。

        与目前的金融系统不同，用户不必通过传统的无数个充满摩擦的步骤来与网络互动并参与其中。

        他们所要做的就是下载或安装一个钱包，就可以开始发送和接收付款，没有任何门槛控制。

        # 创办公司的新方式

        代币也包含了代币化和实现代币经济的想法。

        以目前创办软件公司的情况为例。有人提出了一个想法，但他们需要钱来支持自己开始建设。

        为了得到这笔钱，他们接受风险投资，并给予公司一定比例的股份。

        这种投资立即引入了错位的激励机制，从长远来看，可能与创造最佳的用户体验不太相符。

        而且，如果公司成功了，参与的每个人都需要很长的时间来实现这个价值，往往导致多年的运营没有真正的投资回报。

        • 相反，想象一个令人兴奋的新项目被宣布，解决了一个真正的问题。

        • 任何人都可以从第一天起就参与建设或投资它。

        • 公司宣布推出x代币，并将10%给早期建设者，将10%向公众出售，其余的保留给未来的贡献和项目的融资。

        合作伙伴可以利用他们的代币来决定任务的最终命运的变化，而从风险投资中获得援助的人可以提供一部分资源来带来现金。后来，代币被交付。

        那些相信项目的人可以购买并持有所有权，而那些认为项目正朝着错误方向发展的人可以通过出售他们的股份来表明这一点。

        因为所有的区块链数据都是完全公开的，买方对所发生的事情完全透明。这与购买私人或中心化企业的股票形成鲜明对比，后者的许多事情往往被隐藏起来。

        # 这已经在Web3领域中发生了。

        一个模型是Radicle应用程序（一个分散的GitHub选件），它允许合作伙伴参与管理他们的事业。Gitcoin是另一个允许设计者以数字货币获得补偿的开源问题的弹跳和处理。

        Yearn允许利益相关者参与决策过程并对提案进行投票。

        Uniswap、SuperRare、The Graph、Audius和无数不同的公约和企业都给出了代币作为完成占有、利益和管理的方法。

        DAO（去中心化自治组织）提供了一种建立我们传统上认为的公司的另一种方式，正在获得巨大的动力和来自传统开发商和风险资本公司的投资。

        这种组织是代币化的，通过向更多的利益相关者提供真实、流动和公平的所有权，并以新的和有趣的方式调整激励机制，扭转了组织结构的想法。

        例如，Friends with Benefits是一个由Web3制造商和工匠组成的DAO，大约有一年的历史，截至目前的市值大约为1.25亿美元，最近从a16z得到了1000万美元的投机。

        DAO本身可能会涵盖一整篇文章，但现在，我只想说，我认为它们是建筑产品和（正如我们过去所认为的）公司的未来。

        # Web3中的身份识别是如何工作的

        Web3中的身份工作方式与我们今天的习惯大不相同。在Web3应用程序中，角色经常依赖于与应用程序交互的客户端的钱包地址。

        不像Web2的认证方法，如OAuth或电子邮件+密码（几乎总是要求用户交出敏感的个人信息），钱包地址是完全匿名的，除非用户决定公开链接他们的身份。

        如果用户选择在多个dApp中使用同一个钱包，他们的身份可以在不同的应用程序之间无缝转移，允许他们随着时间的推移建立自己的声誉。

        如果这听起来很复杂，那是因为它很复杂。无可否认，区块链领域仍大多处于测试阶段，但第一批突破性的实施将是爆炸性的，因为它们鼓励用户在财务上加入网络。

        想象一下，如果你真的可以通过为社交媒体做贡献来赚钱。

        比特币的市场份额在过去几年中缓慢下降，虽然我们相信比特币将继续增长，但区块链生态系统的其他部分的快速增长迫在眉睫。

        今天，基于区块链的资源的整个市值是130亿美元；与目前区块链创新可能真正满足的框架价值相比，这是一个调整的错误。

        随着去中心化的区块链协议开始取代主导当前互联网的中心化网络服务，我们将开始看到真正基于互联网的主导地位。未来的互联网将不会是中心化的。
