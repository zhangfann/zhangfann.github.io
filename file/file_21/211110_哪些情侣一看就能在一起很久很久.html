哪些情侣一看就能在一起很久很久

十七亚

1、互相 “瞧得起、看得上”。

不是 “情人眼里出西施” 的带着滤镜那种喜欢，而是打心眼里觉得：这个人在女人（男人）堆里是不错的。

张爱玲说：女人对男人的爱是带有崇拜性的。

同理，男人对女人的（长久的）喜欢，肯定有欣赏的成分。

2、跟对方的同性朋友保持客气、得体的距离。

3、都能看到、珍惜对方的付出。

恋爱中很多问题都是因为只看到自己的付出，认为对方在这段感情里什么都没做而触发矛盾。每个人付出的方式是不一样的。

比如你想吃苹果，但他送了香蕉。你以为他不关注关心你，他以为你不珍惜他的付出。这些事情，一旦聊开了就海阔天空。

4、会吵架，但每次吵完架都会深聊。

不是为了聊这次吵架是谁的责任，而是聊是否有理解错位，哪里有误会。避免下次再因为类似的情况吵架。

双方都听得进对方的解释，也都愿意在这段感情里反思自己的不足。

5、有相对契合的三观。

聊天的内容不局限于吃什么喝什么睡什么去哪玩，会关注、了解彼此的精神世界。长久而亲密的关系，肯定需要精神的大致契合作为基础。

6、能看到、并理解对方的情绪变化。

相处的时间久了，他是高兴的、郁闷的，看他一眼几乎就能知道。这个时候第一反应不是：给我摆脸色干什么？而是询问他发生了什么事，帮他开解。

7、不因为对方比自己进步快，产生自卑情绪而故意从其他方面贬低对方，而是努力让自己也进步。势均力敌的爱情不一定是工作能力上，还有思想、三观等形而上的东西。

8、只有俩人在一块的时候，不存在谁的面子更重要的问题。但是在外人面前，都会为对方的面子着想。

9、都想着让对方高兴。

发自内心的，看到对方开心，自己也高兴。也愿意为他做一些事。不是出于他的要求、指令，而是主动的、贴心地去做。

看到她生气的时候，不是出于赶紧哄好完事，第一反应是看她哭，心里就有说不出的难受。

10、在意他（她）在意的东西。

比如男朋友想有一点私人时间打一会游戏看一会书，那么女生可以暂时放下 “你要时时刻刻陪着我” 的心理。

比如女朋友很注重仪式感，对每个纪念日都很在意。那么男生们就可以在这些节日里准备一些小礼物或者纪念物。

推荐几款在纪念日、哄开心、情人节时可以送女生的礼物。无论多少岁的女人，对礼物没有不心动的。

10、不吝啬表达对彼此的爱意。

说爱是无声的，默默无闻的，我是不太相信。

即使再无声的爱，没有语言，也会有 “爱” 的行动。不会有明显的行动，也会有小细节的关照。

长久的关系肯定需要双方、多方的经营。说实话我没听说过从来没吵过架、从来没闹过矛盾的情侣。****
