# 百度网盘

# 简介

百度网盘是百度推出的一款面向用户的云存储产品, 用户可以将自己的文件上传至网盘中进行保持, 并在需要时进行下载.

# 百度账号登录页
前置条件 有网络 未登录
流程说明 输入账号为手机号
点击下一页至验证码验证页
验证码为6位数字, 60s后可重新发送

# 微信账号登录页

2. 点击百度账号登录至百度账号登录页
3. 点击微信账号登录至微信账号登录页

# 首页

# 首页
前置条件 有网络 已登录
需求说明 点击搜索栏进入搜索页
点击相册进入相册页
点击音频进入音频页
点击视频进入视频页

后置条件 搜索页 相册页 音频页 视频页 传输助手页 打开相应文件
补充说明
# 搜索页
前置条件 有网络 已登录 点击搜索栏
需求说明 进入搜索页后键盘自动弹出
搜索栏内可输入255字符
搜索历史保留最近的4个搜索记录
搜索结果以列表形式展示, 以匹配程度作为排序依据
后置条件 打开相应文件
补充说明
# 传输助手页
前置条件 有网络 已登录 首页-点击传输助手
需求说明 传输助手页分为下载列表, 上传列表, 离线下载
文件都以列表形式展示, 以时间顺序排列
下载列表为用户下载到本地的文件
上传列表为用户上传到系统中的文件
离线下载为用户使用离线下载功能下载的文件
后置条件 打开相应文件
补充说明

# 发现页

# 发现页
前置条件 有网络 已登录
需求说明 发现页分为3栏, 关注, 推荐, 会员
关注栏展示用户的好友近期分享的文件, 以时间顺序排列, 点击文件即可下载

会员栏展示会员专享的可下载文件, 点击文件即可下载

# 个人信息页
前置条件 有网络 我的页-点击用户头像
需求说明 个人信息展示用户的头像, 昵称, 个性签名, 以及开通会员
点击头像进行头像选择, 可以选择相机拍照以及相册选择
点击昵称, 弹出框输入昵称, 昵称长度为12个字符,并且90天能修改一次
点击个性签名, 在弹出框可以输入个性签名, 个性签名为40个字符
点击开通会员, 进入开通会员页
后置条件 开通会员页
补充说明
# 开通会员页
前置条件 有网络 个人信息页-点击开通会员
需求说明 会员包含如下特权: 2T空间, 极速下载, 视频倍速, 畅享内容, 转存数提升, 在线解压, 文件夹备份, 回收站保留
会员包括96元/1年, 30元/3个月, 10元/1个月三档, 点击开通, 跳转至订单支付页, 完成支付
后置条件 返回个人信息, 个人头像处加入会员标识
补充说明
# 设置页

前置条件 有网络 已登录 我的页-点击设置
需求说明 设置页包含个人信息, 账号管理, 备份设置, 网络设置, 文件夹设置, 高级设置,
点击个人信息, 进入个人信息页,
点击自动备份设置, 进入备份设置, 包括相册备份, 文件备份, 音频备份, 微信文件备份, 应用备份,
点击选项后面的按钮, 即可开启/关闭备份
点击网络设置, 可以设置使用手机流量备份照片, 使用手机流量备份视频,使用手机流量播放视频
点击选项后面的按钮, 即可开启/关闭相应网络设置
点击文件夹设置, 可以设置是否显示我的卡包, 显示隐藏空间,显示已购空间
点击高级设置, 可以设置无图模式, 文件名换行显示, 以及清空缓存
后置条件 使相应设置生效
补充说明
